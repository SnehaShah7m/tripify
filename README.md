# Tripify ✈️

**A Platform Where Travelers Shine and Share**

Tripify is a community-driven web platform that allows travelers to share their real-life adventures and inspire others. Users can post their trips with descriptions, images, and YouTube links, while others can browse and discover amazing travel experiences.

---

## 🚀 Features

- User Registration & Login
- Post Travel Experiences (Title, Description, Image URL, YouTube Link)
- Browse All Trips in a clean grid layout
- Responsive Design with modern UI
- User Dashboard (Planned)
- Admin Panel (Planned)

---

## 🛠️ Technologies Used

- **Frontend**: HTML, CSS, Bootstrap, Font Awesome
- **Backend**: PHP
- **Database**: MySQL
- **Local Server**: XAMPP

## 🗄️ Database Setup

1. Open **phpMyAdmin**[](http://localhost/phpmyadmin)
2. Create a new database named `tripify_db`
3. Import the `database.sql` file
4. Update `config.php` if needed (default settings should work with XAMPP)

---

## ▶️ How to Run

1. Copy the `tripify` folder to `C:\xampp\htdocs\`
2. Start **Apache** and **MySQL** from XAMPP Control Panel
3. Open your browser and go to:4. Register a new account or use sample credentials:
